mysql_secure_installation

Before running mysql_secure_installation log on to MySQL as root and set a password for root:

root@server:~# mysql -u root
mysql> alter user 'root'@'localhost' identified with mysql_native_password by 'Str0ng+Pwd';
mysql> commit;
mysql> flush privileges;
mysql> exit;
root@server:~# mysql_secure_installation

Use Str0ng+Pwd when prompted during the mysql_secure_installation app - set a new (stronger) password ...
