Android

  • Install DAVx5 on your mobile phone
  • Create an App Password for your calendar so that DAVx5 can access it: https://myaccount.google.com/apppasswords
  • Configure a profile in DAVx5 with the following information:
    • BaseUrl: https://www.google.com/calendar/dav/${YOUR_ACCOUNT}/events
    • Username: ${YOUR_ACCOUNT}
    • Password: ${APP_PASSWORD} (the one from above)
