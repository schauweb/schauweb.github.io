Windows

Speed up Windows 11 a little

  • Go to Settings -> Accessibility -> Visual effects.
  • Turn off Transparency effects and Animation effects.


Greenshot (Windows 11)

  • Close down Greenshot
  • Go to Settings → Accessibility → Keyboard
  • Make sure that Use the Print screen key to open screen capture is in Off position
  • Run Greenshot
  • Now the Print Screen / PrtSc key works in Greenshot


WSL

Add to WSL profile.rc:

alias sclock='sudo hwclock -s; date'
echo "Time is now: $(date)"
echo
echo "If the above time differs from the computers clock, then please run 'sclock' to synchronize."


Adobe

Cannot print a PDF from within Adobe Reader

  1. Go to Edit → Preferences → Documents and set PDF/A mode to Never
  2. Then go to Edit → Preferences → Security (enhanced) and deselect Enable protected mode at startup

taskkill /F /IM explorer.exe & start explorer

taskkill /F /IM explorer.exe & start


RDP

Disappearing Cursor

The mouse pointer often-times disappears when you move it over a text-field on the remote host. The cure:

  1. Open the Control Panel and select the Mouse applet.
  2. Go to the Pointers tab, select the Text Select item and click Browse...
  3. Select the beam_r.cur item, click Apply and OK.
