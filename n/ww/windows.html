Windows

Speed up Windows 11 a little

  • Go to Settings -> Accessibility -> Visual effects.
  • Turn off Transparency effects and Animation effects.


Calculate SHA256-sum of file

certutil -hashfile ..file.. SHA256


Show reserved ports on Windows

netsh interface ipv4 show excludedportrange protocol=tcp


Disable new context menu

reg.exe add "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32" /f /ve
taskkill /F /IM explorer.exe & start explorer


Restore new context menu

reg.exe delete "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}" /f
taskkill /F /IM explorer.exe & start
